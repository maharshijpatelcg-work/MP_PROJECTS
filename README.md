# 🧮 Simple Counter App

A minimal, fast, and responsive **Counter Application** built with a clean **premium dark UI**.
It helps users easily count, track, and manage numbers with additional control features.

🌐 **Live Demo:** https://counter-mp-app.netlify.app/

---

## ✨ Features

* ➕ Increment counter
* ➖ Decrement counter
* 🔄 Reset counter
* ⚙️ Settings panel
* 🚫 Prevent negative values toggle
* 🔢 Set maximum limit
* 📱 Fully responsive (Mobile + Desktop)
* 🌙 Premium black theme UI

---

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 🧠 How It Works

* The counter value updates dynamically on button clicks.
* Users can:

  * Increase or decrease the value
  * Restrict negative values using toggle
  * Set a custom maximum limit
* Clean UI ensures distraction-free usage.
